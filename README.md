# `create-component-app` templates

A custom set of [create-component-app](https://github.com/CVarisco/create-component-app) templates to help generate React component boilerplate.

## Templates

- `classComponent`: A custom class component (extending reacts component) with CSS modules.
- `presentational`: A stateless component with CSS modules.

## Usage

1. [Install](https://github.com/CVarisco/create-component-app#install) `create-component-app`
2. Clone these templates into project or known root directory
3. Within your project or known root directory, also create a `.ccarc` [config file](https://github.com/CVarisco/create-component-app#you-can-create-a-configuration-file-in-your-current-project-directory)
4. Within the `.ccarc` file set the `templatesDirPath` to the directory of the templates
5. Run `create-component-app --config path_to_your/.ccarc` and follow the prompts, selecting to use a template
